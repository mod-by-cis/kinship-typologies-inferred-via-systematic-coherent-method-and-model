A.2: WSTĘP.

Rzeczą oczywistą jest, że wobec różnorodności modeli rodzinnych niezwykle trudne byłoby skonstruowanie systemu, który mógłby znaleźć zastosowanie inne niż jako ogólny indeks o charakterze pomocniczym. A.3: CEL.

Jaki zatem cel miałby przyświecać tworzeniu takiego systemu? Główną funkcją systemu pozycyjnego jest umożliwienie klasyfikacji typów relacji pokrewieństwa w sposób jednoznaczny i uporządkowany. Dzięki temu możliwe staje się precyzyjne rozpoznanie, czy dana osoba jest np. bratem cioteczno-stryjecznym, czy też bratem wujeczno-ciotecznym. O ile określenie bliskich relacji pokrewieństwa nie nastręcza zwykle trudności, o tyle identyfikacja więzi dalszych wymaga już systematyzującego narzędzia analitycznego. Takie sprawne, strukturalne porządkowanie osób według rodzaju pokrewieństwa okazuje się kluczowe zwłaszcza w genealogii genetycznej — w procesach dedukcji potencjalnych relacji, ustalania stopnia pokrewieństwa oraz w identyfikacji wspólnych fragmentów centy-morganowych dziedziczonych po określonych liniach wspólnych przodków.

A.5: OPIS OBLICZEŃ
  • MAX( ih( i ) ) = total( ih( i ) ) — określa ilość wszystkich węzłów oraz wartość najwyższego indeksu węzła na poziomie danej wysokości.
  • MIN( ih( i ) ) = count( ih( i ) ) — określa ilość węzłów na danym poziomie oraz wartość najniższego indeksu węzła.
  • MIX( ih( i ) ) = MIN( ih( i ) ) + MAX( ih( i ) ) = iLR + iRL —wartość stała dla poziomu, suma MIN i MAX lub suma indeksów LR+RL.

  type NodeConstructor<T> = new (node: number, X?: "XM" | "XW") => T;

  class Node {
    ihINC: number; // iLR: number;
    ihDEC: number; // iRL: number;
    ih: number; // lvX: number;
    MAX: number; // MAX: number;
    MIN: number; // MIN: number;
    MIX: number; // MIX: number;
    iRL: number; // XM: number;
    iLR: number; // XW: number;

    constructor(node: number, X: "XM" | "XW" = "XM") {
      if (typeof node !== "number" || !Number.isInteger(node) || node <= 0) {
        throw new Error("Parametr node musi być dodatnią liczbą całkowitą.");
      }
      const height = Math.floor(Math.log2(node));
      const nodeMAX = 1 * Math.pow(2, height + 1) - 1;
      const nodeMIN = 1 * Math.pow(2, height + 0) - 0;
      const nodeMIX = 3 * Math.pow(2, height + 0) - 1;

      const nodeXM = X == "XM" ? node :  nodeMIX - node;
      const nodeXW = X == "XM" ? nodeMIX - node :  node;
      const sortXM_LR = nodeXM - nodeMIN;
      const sortXW_RL = nodeXW - nodeMIN;

      this.ihINC = sortXM_LR;
      this.ihDEC = sortXW_RL;
      this.ih = height;
      this.MAX = nodeMAX;
      this.MIN = nodeMIN;
      this.MIX = nodeMIX;
      this.iLR = nodeXM;
      this.iRL = nodeXW;
    }

    static ofRange<T extends Node>(
      this: NodeConstructor<T>,
      A: number,
      Z: number,
      X: "XM" | "XW" = "XM",
    ): T[] {
      const OUT: T[] = [];
      for (let i = A; i <= Z; i++) {
        OUT.push(new this(i, X));
      }
      return OUT;
    }

    static ofMulti<T extends Node>(
      this: NodeConstructor<T>,
      R: string,
      X: "XM" | "XW" = "XM",
    ): T[] {
      const RANGE = ((input: string): number[] => {
        const result = new Set<number>();

        for (const token of input.trim().split(" ")) {
          if (!token) continue;

          if (token.includes("..")) {
            const [start, end] = token.split("..").map(Number);
            if (Number.isInteger(start) && Number.isInteger(end)) {
              const step = start <= end ? 1 : -1;
              for (let i = start; step > 0 ? i <= end : i >= end; i += step) {
                result.add(i);
              }
            }
          } else {
            const num = Number(token);
            if (Number.isInteger(num)) {
              result.add(num);
            }
          }
        }

        return [...result].sort((a, b) => a - b);
      })(R);

      const OUT: T[] = [];
      for (const i of RANGE) {
        OUT.push(new this(i, X));
      }
      return OUT;
    }
  }

          } else {

          }
          l--;
        }

C.0-1: ODNIESIENIE DO GENEALOGII.

Zastosowany identyfikator osoby i wykazuje pełną zgodność z konwencją przyjętą w systemie SOSA (szczegóły zostaną przedstawione w kolejnej sekcji).
Osoba wyjściowa — tzw. probant — otrzymuje identyfikator i = 1. Dalsze osoby w drzewie genealogicznym przyporządkowywane są w sposób systematyczny: mężczyznom odpowiadają wartości parzyste (i mod 2 = 0), natomiast kobietom — nieparzyste (i mod 2 = 1).
Parametr ih określa odległość pokoleniową względem probanta: ih = 0 dla probanta, ih = 1 dla jego rodziców, ih = 2 dla dziadków itd.
Reguły przydzielania identyfikatorów są deterministyczne: ojciec osoby i otrzymuje identyfikator 2 × i, matka — 2 × i + 1, zaś dziecko — ⌊i / 2⌋.

Analogiczne zasady obowiązują przy konstruowaniu tzw. linii genealogicznych. W tym kontekście jh = 0 przypisuje się osobie będącej probantem danej linii, tj. najbliższemu znanemu przodkowi, od którego dana linia jest śledzona.
Linie jLL (jL) obrazują przekaz dziedziczny od ojca do dzieci (zarówno synów, jak i córek), co można interpretować jako transmisję nazwiska rodowego bądź autosomalnego DNA o rodowodzie patrylinearno-pokrewieńczym. Natomiast linie jRR (jR) opisują przekaz od matki do dzieci, także w zakresie DNA autosomalnego.

C.0-2: ODNIESIENIE DO GENEALOGII

Dlaczego parametr jh = 0 jest tak znaczący?
Ponieważ linie te, oprócz aspektów genealogicznych, mają również odniesienie do biologii molekularnej: linie ojcowskie odwzorowują dziedziczenie haplogrupy chromosomu Y (przekazywanego wyłącznie z ojca na syna), a linie matczyne — dziedziczenie mitochondrialnego DNA (przekazywanego przez matkę wszystkim dzieciom). Z tego względu kobieta nie może zapoczątkować linii Y-DNA, co czyni poprawne określenie punktu początkowego (jh = 0) niezbędnym dla zachowania spójności reprezentacji dziedziczenia.
Każda wartość jh oznacza zatem liczbę pokoleń, które dzielą daną osobę od probanta danej linii — nie zaś od protoplasty linii jako takiej.

Co najistotniejsze, wszystkie wspomniane informacje — zarówno ih, jak i jh — są zawarte implicytnie w strukturze identyfikatora i. Przy zastosowaniu odpowiednich wzorów możliwe jest ich dynamiczne wyprowadzenie, bez konieczności jawnego zapisu.

C.2-2: HISTORIA (von Aitzing (Eytzing))

-> 1590 <- Prekursorem systemu był Michael Freiherr von Aitzing (Eytzing) (ur. ok. 1530 r. na zamku Schrattenthal [48°42'52"N 15°54'21"E], zm. w 1598 r. w Bonn [50°44'N 7°06'E]). Ten dolnoaustriacki arystokrata, wyznania katolickiego, był erudytą, pisarzem oraz pionierem w dziedzinach genealogii i wczesnego dziennikarstwa. Około 1550 roku podjął studia prawnicze na Uniwersytecie w Leuven (Belgia), gdzie rozwijał również zainteresowania w zakresie kartografii, matematyki, historii i filologii [ISBN 978-3-902684-35-6, s. 50]. Już w 1553 roku opublikował swoje pierwsze dzieło, podręcznik retoryki pt. Artis oratiae tabulae. Aitzing odbywał liczne podróże i cechował się biegłą znajomością wielu języków starożytnych i nowożytnych, takich jak greka, łacina, hebrajski, hiszpański, włoski i flamandzki [ISBN 3-7616-0858-6, s. 481]. Po ukończeniu studiów powrócił do Wiednia, gdzie cesarz Ferdynand I mianował go "nadwornym sługą"; służył również jego następcom, Maksymilianowi II i Rudolfowi II, jako radca i szambelan do 1576 roku. W 1563 roku Aitzing, będący małżonkiem przedstawicielki rodu Fuggerów (wpływowej szwabskiej rodziny tkacko-kupieckiej z Augsburga), uczestniczył w Soborze Trydenckim w imieniu cesarza, a w 1566 roku wziął udział w sejmie augsburskim. W 1568 roku był naocznym świadkiem egzekucji Lamorala, hrabiego Egmont, w Brukseli, gdzie, jak się przypuszcza, przebywał w charakterze "tajnego obserwatora" dworu austriackiego ["Allgemeine Deutsche Biographie", Band 5, Duncker & Humblot, Leipzig 1877, s. 777]. Następnie sam został tamże uwięziony, prawdopodobnie z przyczyn politycznych. Po uwolnieniu, w 1581 roku osiadł w Kolonii, gdzie kontynuował działalność pisarską. Aitzing opracował kartograficzny wizerunek Ziemi Świętej, który został wyryty w miedzi przez Fransa Hogenberga w 1582 roku. Rozgłos przyniosło mu łacińskojęzyczne dzieło Leo Belgicus, opublikowane po raz pierwszy w 1583 roku. Kluczowym dla genealogii okazało się jednak wydanie w Kolonii w 1590 roku pracy Thesaurus principum hac aetate in Europa viventium, quo progenitores eorum tam paterni quam materni simul ac fratres et sorores inde ab origine reconduntur, usque ad annum a Christo nato. W publikacji tej po raz pierwszy zastosowano drukowaną numerację przodków według systemu, który w późniejszym okresie zyskał miano metody Sosa-Stradonitza. Mimo sukcesów publicystycznych, Aitzing doświadczał w Kolonii znacznych trudności materialnych. Ostatecznie przyjął zaproszenie elektora i arcybiskupa Bawarii, Ernsta, udając się do Bonn, gdzie zmarł w 1598 roku jako jego historiograf.

C.2-3: HISTORIA (Sosa)

-> 1676 <- Kolejną znaczącą postacią był Jerónimo de Sosa (ur. w XVII w., zm. w 1711 r.). Ten hiszpański franciszkanin, profesor teologii w królewskim klasztorze Santa Maria la Nova w Neapolu, był również cenionym genealogiem. W swoim opublikowanym w 1676 roku dziele genealogicznym Noticia de la Gran Casa de los marqueses de Villafranca, de Sosa świadomie zastosował i spopularyzował metodę numeracji przodków, którą prawie sto lat wcześniej zainicjował Michael Freiherr von Aitzing.

C.2-4: HISTORIA (Galton)

-> 1883 <- Wkład w rozwój myśli genealogicznej, choć w szerszym kontekście swoich badań, wniósł Sir Francis Galton (ur. w 1822 r. w Sparbrook, Birmingham, zm. w 1911 r. w Haslemere, Surrey). Ten brytyjski polihistor, pionier eugeniki (twórca samego terminu), antropolog, geograf, eksplorator, wynalazca, meteorolog, statystyk, psycholog i genealog, był przyrodnim kuzynem Karola Darwina. Dorobek pisarski Galtona obejmuje ponad 340 artykułów i książek. Wprowadził on fundamentalne dla statystyki koncepcje korelacji i regresji do średniej. Jako pierwszy zastosował metody statystyczne do badania różnic indywidualnych oraz dziedziczenia inteligencji, a także wprowadził kwestionariusze i ankiety do gromadzenia danych o społecznościach ludzkich, które były mu niezbędne do prac genealogicznych, biograficznych oraz badań antropometrycznych. Był pionierem eugeniki, formułując ten termin i jego założenia. Jego książka Hereditary Genius (1869) stanowiła pierwszą próbę naukowego, opartego na metodach nauk społecznych, zbadania dziedziczności geniuszu. Jako badacz umysłu ludzkiego stworzył podwaliny psychometrii (nauki o pomiarze zdolności umysłowych), psychologii różnic indywidualnych oraz leksykalnej hipotezy osobowości. Opracował metodę klasyfikacji odcisków palców, która okazała się niezwykle użyteczna w kryminalistyce. Będąc inicjatorem meteorologii naukowej, opracował pierwszą mapę pogody, zaproponował teorię antycyklonów i jako pierwszy ustanowił kompletny zapis krótkoterminowych zjawisk klimatycznych w skali europejskiej. Nie piastował stałej profesury uniwersyteckiej, a większość badań prowadził samodzielnie. Jego liczne zasługi zostały formalnie uznane, gdy w wieku 87 lat otrzymał tytuł szlachecki (Sir). Galton ukuł termin "eugenika" (rozumiana jako sztuczna selekcja mająca na celu doskonalenie rasy ludzkiej) w 1883 roku, a wiele ze swoich obserwacji i wniosków, w tym o charakterze genealogicznym, zawarł w książce Inquiries into Human Faculty and Its Development. Jego prace, promujące systematyczne podejście do badania dziedziczności, pośrednio przyczyniły się do zainteresowania metodami organizacji danych genealogicznych.

C.2-5: HISTORIA (Kekulé von Stradonitz)

-> 1898 <- Ostateczne skodyfikowanie i szerokie spopularyzowanie systemu przypisuje się Stephanowi Kekulé von Stradonitz (ur. w 1863 r. w Gandawie, zm. w 1933 r. w Berlinie). Ten niemiecki prawnik, prywatny uczony, heraldyk i genealog, pochodził z czeskiej rodziny szlacheckiej; w 1895 roku został nobilitowany w Niemczech przez cesarza Wilhelma II (przydomek "von Stradonitz" nawiązuje do Stradonic koło Kłodna, miejsca pochodzenia jego patrylinearnych przodków). W 1898 roku Kekulé von Stradonitz opublikował swoją interpretację metody Aitzinga (Eytzingera), Sosy i, być może niezależnie, koncepcji zbliżonych do Galtona, w monumentalnym dziele Ahnentafel-Atlas. Ahnentafeln zu 32 Ahnen der Regenten Europas und ihrer Gemahlinnen (Berlin: J. A. Stargardt, 1898–1904). Publikacja ta, zawierająca 79 tablic genealogicznych władców Europy i ich małżonek, w decydujący sposób przyczyniła się do ugruntowania omawianej metody jako najpowszechniej stosowanej w naukach genealogicznych. Od tego czasu system ten jest szeroko znany jako metoda Sosa-Stradonitza lub, z języka niemieckiego, Ahnentafel.

C.3-1: OGRANICZENIA (brak redukcji)

Za potencjalne ograniczenie można by uznać brak redukcji numeracji w przypadku tzw. ubytku przodków (sytuacji, gdy ta sama osoba pojawia się w wielu liniach genealogicznych). Choć na pierwszy rzut oka może to wydawać się problemem (np. pozorną redundancją), takie powielenie numeracji jest w rzeczywistości celowym założeniem systemu. Ten sam przodek, występując w różnych gałęziach drzewa genealogicznego, pełni w każdej z nich unikalną funkcję w kontekście danej ścieżki ancestralnej. Dlatego też odrębne, choć zwielokrotnione, oznaczenie każdego wystąpienia przodka jest istotne dla zachowania ciągłości, integralności i analitycznej przejrzystości każdej linii genealogicznej.
